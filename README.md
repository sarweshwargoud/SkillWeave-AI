# 🚀 SkillWeave AI  
### AI-Powered Learning Path & YouTube Playlist Generator

SkillWeave AI is an intelligent AI/ML system that transforms scattered YouTube educational videos into a **structured, continuous, and personalized learning roadmap**.

Built as part of the **Edunet Foundation AI & ML Internship**, this project uses advanced technologies such as NLP, LLMs, and Speech Processing to curate complete learning paths automatically.

---

## 📌 Problem We Solve.
Students rely on YouTube to learn skills, but face
- ❌ Unstructured and scattered videos  
- ❌ Incomplete playlists  
- ❌ Repeated or skipped topics  
- ❌ Sudden jumps in learning flow  
- ❌ Difficulty understanding unfamiliar accents  

---

## 💡 Our Solution
SkillWeave AI automatically:
- ✅ Generates a structured syllabus  
- ✅ Builds curated YouTube playlists  
- ✅ Ensures smooth topic continuity  
- ✅ Matches instructors by accent & style  
- ✅ Filters high-quality educational content  

---

## 🧠 Key Features
- AI-based video topic analysis  
- Transcript understanding using LLMs  
- Topic continuity verification  
- Accent-aware teacher grouping  
- Personalized learning paths  
- Responsive modern UI
