## Setup

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. Set your API keys in `.env`.
4. Run: `uvicorn main:app --reload`

### Frontend
1. `cd frontend`
2. `npm install` (if not already done)
3. `npm run dev`
